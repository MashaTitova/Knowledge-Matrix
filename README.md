# Knowledge-Matrix
## Приложение для интеллектуального квиза
### Задача: ответить на все вопросы каждой из 6 категорий
### Входные данные:
- Файл с вопросами по 22 категориям и 3 уровням сложности
- Файл с изображениями для кнопок категорий
### Как запустить через Visual Studio
- Склонируйте репозиторий 
- Откройте файл Knowledge-Matrix.slnx
- Запустите программу
### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub. Ссылка: https://github.com/MashaTitova/Map-of-ecological-zones.git
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с тем же именем, что и сам репозиторий. Если нужно указать другую папку, в конце команды для клонирования следует указать её название: git clone <ссылка на репозиторий> <папка>
### Пример использования
- Пользователь изучает инструкцию, которая появляется при нажатии на кнопку "Об игре"
- Пользователь нажимает на кнопку "Доказать свою эрудицию"
- Пользователь выбирает катогорию вопроса
- Пользователь выбирает уровень сложности вопроса
- Пользователь дает ответы на вопросы, нажав на кнопку с одним из вариантов ответов
### Особенности программы
- Понятный и приятный графический пользовательский интерфейс, удобное меню, удобное управление
- Взаимодействие с пользователем через появляющиеся сообщения
- Простая текстовая анимация
### Требования
- .NET 4.8 или новее
- Visual Studio 2022 (или любая среда с поддержкой C#. Версия языка 14.0 и выше)
